## Cybersecurity Act (2019/881)

**Consolidated CELEX:** `02019R0881-20250204`
**Imported from:** Base CELEX `32019R0881`

### M1: Regulation (EU) 2025/37

| Property | Value |
|----------|-------|
| **Effective** | 4 February 2025 |
| **Subject** | Managed security services |
| **Impact** | Substantive — adds new service category |

**Key Changes:**
- Adds "managed security services" as a new category
- New Article 49a on managed security services
- New Article 51a on European cybersecurity certification schemes
- Modifications to Articles 46, 47, 49, 52, 53, 54

**Why not applied:**
- Consolidated Formex not available on EUR-Lex
- Would require manual paragraph-level editing
- Content note added to markdown metadata instead

**Action:** Users clicking "View on EUR-Lex" see the consolidated version with these amendments.
