## EUR-Lex Anchor Pattern

| Target | Anchor Format | Example |
|--------|---------------|---------|
| Article N | `#art_N` | `#art_5` |
| Article N, paragraph M | `#NNN.MMM` (zero-padded) | `#005.001` |
| Recital N | `#rct_N` | `#rct_26` |
| Chapter (Roman) | `#cpt_N` | `#cpt_II` |

**Limitation**: Points `(a)`, `(b)` and definition lists `(1)`, `(2)` do NOT have individual anchors.

## Implementation Notes

1. **EUR-Lex Base URL**: `https://eur-lex.europa.eu/legal-content/EN/TXT/HTML/?uri=CELEX:{CELEX}`
2. **Article anchor**: Append `#art_N` for Article N
3. **Paragraph anchor**: Append `#NNN.MMM` (zero-padded to 3 digits each)
4. **Recital anchor**: Append `#rct_N`
